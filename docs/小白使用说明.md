# 小红书创作者工具箱：小白使用说明

这套工具适合两类人：

1. 想把小红书帖子保存成结构化资料的人。
2. 想做选题整理、内容分析、封面生成的小红书内容创作者。

如果你完全没接触过 Claude Code、skill、cookies，也没关系。按下面一步一步来就行。

## 你能用它做什么

- 保存单条小红书帖子到本地 Markdown
- 批量保存多条帖子
- 分析你已经保存的内容，提炼选题和结构
- 生成小红书封面图
- 用本地服务做“一键剪藏”

## 方式一：用 Claude Code 里的 skill

这是最推荐的方式，功能最完整。

### 第 1 步：安装插件

在 Claude Code 里执行：

```bash
/plugin marketplace add chenxiachan/xhs-claude-skills
/plugin install rednote-to-obsidian@chenxiachan-xhs-claude-skills
```

### 第 2 步：准备 cookies

1. 用 Chrome 打开 `https://www.xiaohongshu.com`
2. 确认你已经登录
3. 按 `F12` 打开开发者工具
4. 切到 `Console`
5. 粘贴下面这段代码并回车

```javascript
copy(JSON.stringify(document.cookie.split('; ').map(c => {
  const [name, ...rest] = c.split('=');
  return {
    name,
    value: rest.join('='),
    domain: '.xiaohongshu.com',
    path: '/',
    expires: Date.now() / 1000 + 86400 * 30,
    size: name.length + rest.join('=').length,
    httpOnly: false,
    secure: false,
    session: false,
    priority: 'Medium',
    sameParty: false,
    sourceScheme: 'Secure',
    sourcePort: 443
  };
})))
```

6. 新建一个文件，命名为 `~/cookies.json`
7. 把刚才复制的内容粘进去并保存

### 第 3 步：开始使用

常用命令如下：

```bash
/xhs https://www.xiaohongshu.com/explore/...
/xhs-batch <多个链接>
/xhs-analyze
/xhs-cover 爆款标题文案
```

### 第 4 步：查看输出位置

默认会保存到：

```text
~/Documents/Obsidian Vault/xhs
```

如果你想换目录，就修改 skill 里的默认路径说明，或者在本地服务里设置环境变量。

## 方式二：用本地服务一键剪藏

如果你不想每次都进 Claude Code，可以直接开一个本地服务。

### 第 1 步：启动服务

```bash
cd service
python3 server.py
```

启动成功后，浏览器访问：

```text
http://127.0.0.1:7895/health
```

如果看到 `{"ok": true}`，说明服务正常。

### 第 2 步：发送剪藏请求

你可以直接在终端测试：

```bash
curl "http://127.0.0.1:7895/clip?url=https://www.xiaohongshu.com/explore/..."
```

### 第 3 步：做成浏览器书签

把下面整段保存成浏览器书签的网址：

```javascript
javascript:(function(){fetch('http://127.0.0.1:7895/clip',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({url:location.href})}).then(r=>r.json()).then(data=>alert(data.success?'已保存：'+data.file:'失败：'+(data.error||'未知错误'))).catch(err=>alert('请求失败：'+err));})();
```

之后你打开任何一条小红书帖子，点一下这个书签，就能自动保存。

## 常见问题

### 1. 提示 cookies 失效

重新按上面的方式导出一次 `cookies.json` 即可。

### 2. 没有保存成功

先检查三件事：

1. 你是否已经登录小红书
2. `~/cookies.json` 是否存在
3. 输出目录是否有写权限

### 3. 视频转录为什么没有出来

本地服务版本主要负责快速剪藏。更完整的视频转录建议通过 Claude Code 里的 `/xhs` skill 来做。

## 隐私说明

- 你的 cookies 只保存在本机
- 仓库默认不会上传 cookies、日志、缓存和本地输出
- 不要把你自己的笔记库、截图、导出的 cookies 一起提交到 git
